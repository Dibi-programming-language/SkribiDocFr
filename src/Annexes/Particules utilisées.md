## Informations

| Particule | Utilisation en Dibi   | Utilisation en Skribi     |
| --------- | --------------------- | ------------------------- |
| `nu-`     | Origine (lieu)        | Boucle for bornée         |
| `bi-`     | Destination (lieu)    | Boucle for bornée         |
| `ci-`     | Temps                 | Boucle while              |
| `da-`     | Thème                 |                           |
